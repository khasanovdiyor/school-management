## 1. Clone the repo

```bash
$ git clone https://github.com/khasanovdiyor/school-management
```

## 2. Installation

```bash
$ npm install
```

## 3. Make sure to have postgres running on port 5432 if you don't have it run:

```bash
$ docker compose up -d
```

## 4. Set/Change env vars

## 5. Run the app

```bash
# development
$ npm run start
```
